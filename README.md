# Phạm Lê Ngọc Sơn CRM (CRUD)
Full-stack CRM built for Phạm Lê Ngọc Sơn using Angular and Spring Boot REST APIs.

![Status started](https://img.shields.io/badge/Status-Started-blue.svg)
![Open Source Love](https://badges.frapsoft.com/os/v1/open-source.svg?v=102)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

### Used Technologies & Tools
* Front-end
  * Angular 8 with Bootstrap UI.
* Back-end
  * Spring Boot 2.x (REST API) [Java 8].
  * Spring Security (Authentication & JWT).
  * Maven.

### Demo Access
